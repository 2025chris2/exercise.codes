垂直对齐:垂直对齐一解决图片底部空白缝隙问题 基线:浏览器行内元素(行内块元素)排版中存在用于对齐的基线(baseline) 所有的浏览器都有四条线：顶线(top)中线(middle)基线(baseline)底线(bottom) 因为图片默认和文字基线对齐,所有我们在切的时候图片的下面两个角并不能切的和上面两个角一样！ 解决方案: 1.把图片转换为块级元素。 2.设置图片对齐方式不是基线对齐即可。(只要不是baseline就可以解决) img { vertical-align: top; } baseline 默认,基线对齐 top 顶部对齐 middle 中部对齐 bottom 底部对齐
